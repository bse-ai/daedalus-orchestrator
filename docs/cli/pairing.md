# `forge-orchestrator pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
forge-orchestrator pairing list whatsapp
forge-orchestrator pairing approve whatsapp <code> --notify
```
